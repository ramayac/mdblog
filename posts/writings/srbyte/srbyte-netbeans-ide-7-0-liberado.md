El popular entorno de desarrollo llega a su septima entrega, brindando soporte temprano para las propuestas y especificaciones de Java SE 7 y JDK 7 (developer preview). Ademas de incluir soporte para productos de Oracle: como Oracle WebLogic y Oracle Database.

El dia de hoy (19 Julio 2011) el comite ejecutivo de la JCP (Java Community Process) aprobó de forma definitiva la versión 7 de Java SE, con 15 votos a favor, y 1 en contra (de Google, con respecto a un["problema" de licenciamiento](https://www.jcp.org/en/jsr/results?id=5225)).

Los siguientes JSR (Java Specification Request) fueron aprobados con unanimidad:

- NIO2 (Nuevo API para I/O): [https://www.jcp.org/en/jsr/results?id=5219](https://www.jcp.org/en/jsr/results?id=5219)
- InvokeDynamic (soporte para lenguajes dinamicos): [https://www.jcp.org/en/jsr/results?id=5221](https://www.jcp.org/en/jsr/results?id=5221)
- Project Coin (Mejoras a la sintaxis de Java): [https://www.jcp.org/en/jsr/results?id=5223](https://www.jcp.org/en/jsr/results?id=5223)

Ahora solo falta esperar nueve dias aproximadamente para que esta nueva versión salga a "producción", es decir la versión de los binarios "comerciales" de Oracle, que por cierto están basados en el OpenJDK ;)

Para que se emocionen (¡ja!) los invito a que vean el vídeo de [Introducción a Java 7](https://www.oracle.com/us/corporate/events/java7/index.html) (webcast del Java 7 Launch Day).

¡Saludos!
